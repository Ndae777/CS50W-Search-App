# CS50W Search App

## 📌 Description
This project is a front-end implementation of a Google Search interface, built as part of the CS50 Web Programming course. It replicates core Google search functionalities, including standard search, image search, and advanced search.

## 🚀 Features
- Google search page clone
- Image search functionality
- Advanced search form with multiple query parameters
- Clean and structured UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap
